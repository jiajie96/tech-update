# AI Agents Weekly Update — May 7, 2026

## Highlights

- **Microsoft Agent 365 Goes GA**: Microsoft's dedicated governance platform for enterprise AI agents launched on May 1 at $15/user/month, with cross-cloud registry sync covering AWS Bedrock and Google Cloud.
- **Google Unveils Gemini Enterprise Agent Platform**: At Cloud Next 2026, Google announced a unified agent-building platform replacing Vertex AI, featuring the Agent Development Kit (ADK) and visual Agent Studio.
- **Linux Foundation's Agentic AI Foundation Gains Momentum**: MCP has crossed 97 million monthly SDK downloads and adoption by every major AI provider, while the protocol's 2026 roadmap focuses on transport scalability and enterprise readiness.

## Detailed Updates

### Microsoft Agent 365 Generally Available
Microsoft Agent 365 launched on May 1 as a dedicated console for deploying, managing, and securing autonomous AI agents at enterprise scale. Priced at $15 per user per month and separate from Copilot, Agent 365 now includes public preview of registry sync with AWS Bedrock and Google Cloud, enabling IT teams to discover, inventory, and govern agents across multi-cloud environments. Microsoft is positioning agents as the "next operating layer for work," shifting Copilot from a sidebar helper to an orchestrator of autonomous digital workers.

### Google's Gemini Enterprise Agent Platform
Google announced the Gemini Enterprise Agent Platform at Cloud Next 2026, designed to replace Vertex AI as Google's primary enterprise AI development environment. The platform bundles the Agent Development Kit (ADK), a code-first environment supporting graph-based multi-agent networks, with Agent Studio, a low-code visual interface for designing agent logic without writing code. The move consolidates Google's fragmented agent tooling under a single roof.

### ServiceNow and NVIDIA Extend Agent Collaboration
At ServiceNow Knowledge 2026, ServiceNow and NVIDIA extended their partnership to deliver governed autonomous agents for enterprises. Project Arc connects natively to the ServiceNow AI Platform through Action Fabric, bringing governance, auditability, and workflow intelligence to every action the desktop agent takes. ServiceNow is implementing a "tollgate" consumption-based pricing model based on the volume of work AI agents perform.

### MCP Protocol Ecosystem Expansion
The Model Context Protocol has crossed 97 million monthly SDK downloads (Python + TypeScript combined) and been adopted by every major AI provider including Anthropic, OpenAI, Google, Microsoft, and Amazon. More than 10,000 published MCP servers now cover everything from developer tools to Fortune 500 deployments. The official 2026 roadmap prioritizes transport scalability, agent communication, governance maturation, and enterprise readiness.

### Coder Launches Self-Hosted Agent Platform
Coder Technologies announced the beta release of Coder Agents, giving enterprises a way to deliver agent-driven developer workflows without sending source code outside their network perimeter. The entire agent system runs on customer-owned infrastructure, addressing a key concern for organizations wary of sending proprietary code to third-party AI services.

### Enterprise AI Agent Adoption Surges
AI investment activity is on track to surpass $650 billion annually as enterprise adoption accelerates. As of mid-2026, 54% of enterprises have integrated AI agents into core operations as autonomous systems executing workflows, processing documents, monitoring compliance, and coordinating decisions across business functions — up from 22% adoption in 2025.

### Agent Security Becomes a Priority
With autonomous software moving from demos into real business operations, prompt injection attacks — where malicious actors trick AI into performing unauthorized actions — have become a major focus for cybersecurity firms in 2026. Companies are increasingly recognizing that agents require the same security governance as any semi-independent worker with permissions, memory, and access to sensitive data.
