# AI Agents Weekly Update — May 9, 2026

## Highlights

- **Five Eyes Issue Urgent Agentic AI Safety Guidance**: Security agencies from the US, UK, Canada, Australia, and New Zealand jointly released guidance titled "Careful Adoption of Agentic AI Services," warning that rapid rollouts are too risky and recommending slow deployment starting with low-risk tasks while keeping humans in control.
- **Coder Agents Launch for Self-Hosted AI Coding**: Coder Technologies released Coder Agents in beta, a fully self-hosted, model-agnostic AI coding agent that keeps planning, orchestration, and execution entirely within customer-controlled infrastructure.
- **Microsoft Agent 365 Goes Live**: Microsoft activated Agent 365, a system of autonomous AI agents that independently perform enterprise tasks including file management, communications, and procurement workflows within the Microsoft 365 environment.

## Detailed Updates

### Five Eyes Agentic AI Security Warning
Cybersecurity and intelligence agencies from five nations jointly released guidance urging organizations to deploy AI agents slowly and carefully. The guidance specifically recommends starting with low-risk tasks, keeping humans in control of critical decisions, and establishing robust monitoring before expanding agent autonomy. This follows disclosures of six exploits targeting credentials, tokens, and permissions in major coding agents between 2025 and 2026.

### Coder Agents: Self-Hosted AI Coding
Coder Technologies announced the beta release of Coder Agents, positioning it as the enterprise-grade alternative to cloud-dependent AI coding tools. The platform is model-agnostic, allowing organizations to plug in any AI model while keeping the entire agent system on their own infrastructure. This addresses growing enterprise concerns about code security and IP exposure in cloud-hosted coding agents.

### Snyk-Claude Security Partnership
Anthropic's Claude models are now integrated into the Snyk AI Security Platform, bringing advanced reasoning capabilities to automated security scanning and vulnerability detection. The integration is available to joint customers now with expanded access rolling out through 2026.

### Opsera-Cursor DevSecOps Integration
Opsera announced a partnership with Cursor to embed DevSecOps Agents directly into Cursor's IDE, bridging developer velocity with enterprise security. This reflects the broader trend of security tooling meeting developers where they already work rather than adding separate compliance steps.

### Microsoft Agent 365 Enterprise Deployment
Microsoft launched Agent 365, deploying autonomous AI agents that can independently update shared files, manage communications, and streamline procurement workflows across the Microsoft 365 ecosystem. The system is designed for enterprise collaboration with built-in governance controls.

### Salesforce and Cloudflare Activate Autonomous Agents
Salesforce opened its agent platform so AI agents can execute workflows directly without human intervention, while Cloudflare enabled agents to deploy applications autonomously on its infrastructure. Both represent major steps toward production-grade autonomous agent deployment.

### NVIDIA-ServiceNow Governed Enterprise Agents
At ServiceNow Knowledge 2026, NVIDIA and ServiceNow expanded their collaboration to deliver governed autonomous agents to enterprises, spanning from employee desktops to AI factory operations.

### Adoption Gap Persists
Despite 79% of companies reporting AI agent adoption, only 2% have achieved full deployment, with 55% of leaders citing concerns about reliability and errors as the primary barrier to scaling.

### AI Agent Protocol Maturation
MCP has crossed 97 million monthly SDK downloads and been adopted by every major AI provider. The Linux Foundation's Agentic AI Foundation now stewards MCP, Block's goose, and OpenAI's AGENTS.md as foundational open projects for the agent ecosystem.
