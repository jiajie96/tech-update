# AI Agents Weekly Update — May 2, 2026

## Highlights

- **Microsoft Agent 365 Goes GA**: Microsoft's enterprise agent management platform launched generally available on May 1 at $15/user/month, with cross-cloud governance spanning AWS Bedrock and Google Cloud.
- **Human Scientists Still Outperform AI Agents on Complex Tasks**: A Nature study found that human researchers significantly outperform the best AI agents on multi-step scientific reasoning tasks, tempering expectations around full autonomy.
- **GameMaker Integrates AI-Powered "Vibe Coding"**: GameMaker's 2026 platform update adds AI-assisted game development workflows, signaling the spread of agentic coding tools beyond traditional IDEs.

## Detailed Updates

### Microsoft Agent 365 Generally Available
Microsoft Agent 365 launched as generally available on May 1, 2026, priced at $15 per user per month for commercial customers. The platform serves as a control plane for governing AI agents across Windows, Azure, and multicloud environments. A key announcement was the public preview of Agent 365 registry sync with AWS Bedrock and Google Cloud, enabling IT teams to discover, inventory, and manage agent lifecycles across cloud providers. This cross-platform approach positions Microsoft as the default governance layer for enterprise agent deployments.

### Nature Study: Humans Outperform AI Agents on Complex Tasks
A study published in Nature found that human scientists significantly outperform the best AI agents on complex, multi-step tasks requiring deep domain expertise and creative problem-solving. While AI agents excel at well-defined subtasks and information retrieval, the research highlights that fully autonomous scientific discovery remains a distant goal. The findings are a reality check for companies betting on near-term agent autonomy for high-stakes research workflows.

### MCP Adoption Accelerates Across Platforms
The Model Context Protocol (MCP), originally developed by Anthropic as an open standard for connecting AI models to tools and data sources, continues to gain enterprise traction. Microsoft 365 Copilot now supports building declarative agents with MCP endpoints via the Microsoft 365 Agents Toolkit in VS Code. The toolkit provides a guided experience to connect agents to MCP servers, enabling developers to build custom tool integrations without deep infrastructure work.

### GameMaker Embraces Agentic AI for Game Development
On April 30, GameMaker announced a major platform update integrating AI-powered coding workflows. The 2026 GMRT update adds AI-assisted "vibe coding" capabilities, GM-CLI tools, and C#/JS support. The integration allows game creators to build and iterate using natural language prompts, reflecting how agentic AI tools are expanding from professional software engineering into creative development environments.

### Enterprise Agent Adoption Crosses the Chasm
Industry reports indicate that agentic AI has moved from experimental to production deployments across software engineering, finance, healthcare, and business operations in 2026. Unlike chatbots, production agents operate in continuous plan-act-observe-adapt loops until tasks complete. The shift is driven by improvements in tool-use reliability, longer context windows, and better error recovery — though governance and safety remain top concerns for enterprise adopters.

### GDC 2026 Survey: Agent Tool Use Among Developers
The GDC 2026 State of the Game Industry report revealed that over one-third (36%) of game industry professionals now use generative AI tools in their workflows. However, adoption is uneven: 58% of publishing, support, and marketing professionals use AI tools compared to just 30% at game studios. The data suggests agentic tools are gaining ground faster in operational roles than in core creative development.
