# AI Agents Weekly Update — April 29, 2026

## Highlights

- **MCP Hits 164M Monthly Downloads**: The Model Context Protocol ecosystem continues explosive growth, with 2,400+ servers now in production and major new integrations from Cloudflare, Google Colab, and Microsoft Fabric.
- **Cequence Launches Agent Personas**: The first infrastructure-level privilege scoping system for autonomous AI agents, giving enterprises granular control over what agents are permitted to do down to specific tool calls.
- **Enterprise Agent Adoption Accelerates**: 40% of enterprise apps are projected to have AI agents by year-end 2026, up from 5% in 2025, though only 10% of organizations have scaled agents beyond pilot mode.

## Detailed Updates

### MCP Ecosystem Reaches Critical Mass
The MCP Dev Summit in New York City (April 2-3) drew 1,200 attendees across 95 sessions. The protocol now has 164 million monthly downloads and over 2,400 servers, representing 10-100x more developer momentum than any competing standard. Cloudflare launched a Code Mode MCP server that reduces token usage for API interactions by 99.9% — from 1.17M tokens to roughly 1,000 tokens across 2,500+ endpoints. Google released an open-source Colab MCP Server enabling agents to offload compute-intensive tasks to cloud notebooks. Microsoft integrated MCP into its Fabric data platform, positioning it as the connective layer for an "AI-native operating system."

### Cequence Agent Personas for Security
Cequence Security announced general availability of Agent Personas in its AI Gateway on April 28, introducing the industry's first infrastructure-level privilege scoping for autonomous agents. The system lets enterprises define granular permissions for what AI agents can access and execute, addressing a critical gap as agent deployments scale.

### Novita AI Sandbox for Agent Isolation
Novita AI launched Novita Sandbox on April 28, providing system-level isolation with sub-200ms startup times for autonomous agent deployments. The sandbox is designed to secure agent systems like OpenClaw and Hermes Agent by containing their execution environments.

### Zero Networks AI Segmentation
Zero Networks launched AI Segmentation on April 21, offering identity-based network controls specifically designed to govern autonomous AI agents and prevent AI-driven lateral movement across corporate networks. The product reflects growing concern about agent security in enterprise environments.

### NVIDIA and Adobe Partner on Creative Agents
NVIDIA and Adobe announced a partnership with WPP to build autonomous creative agents that can handle end-to-end content production workflows. The agents leverage NVIDIA's inference infrastructure and Adobe's creative tools to generate marketing and advertising content at scale.

### Enterprise Agent Market Shifts
Market data shows Anthropic now commands 40% of enterprise LLM API spend, while OpenAI's share has declined to 27% from 50% in 2023. Automation Anywhere reported that AI agents auto-resolve over 80% of IT support requests, cutting IT service management costs by up to 50%.

### Google Rebrands to Agent Platform
Google rebranded its AI platform as the Gemini Enterprise Agent Platform, offering 200+ foundation models and enterprise governance in a unified package — a clear signal that agents, not just models, are the primary product.
