## 🗓️ Schedule

Updates are published **every Friday at 8:00 AM** (local time).

Each weekly report is saved as `YYYY-MM-DD.md` inside the relevant category folder.

## 📖 How It Works

This repository is automatically maintained by a scheduled Claude task that:
1. Searches the internet for the latest news and developments in each category
2. Curates and summarizes the most significant updates
3. Saves reports to the appropriate category folders
4. Updates this README with links to the latest reports
5. Commits and pushes changes to GitHub
